CATATAN PENTING

Folder ini berisi source code final aplikasi REST API Client.
Source code utama TIDAK dipindah ke struktur aneh; tetap memakai struktur Electron asli:
- src/main
- src/preload
- src/renderer

node_modules sengaja tidak dimasukkan karena bukan source code dan sebaiknya dibuat ulang dengan npm install.

Cara menjalankan:
1. npm install --legacy-peer-deps
2. npm run dev

Build sudah dicek di environment ChatGPT dengan npm run build dan berhasil.
